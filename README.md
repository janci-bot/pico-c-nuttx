# 🚀 Learn C with Raspberry Pi Pico → NuttX

Learning project focused on **C programming**, **embedded development**, and **NuttX RTOS** using **Raspberry Pi Pico 2 WH**.

---

## 🎯 Project Goals

* Learn **C language** for embedded development
* Work with **Raspberry Pi Pico 2 WH**
* Understand **embedded architecture**
* Learn **RTOS concepts**
* Run **NuttX on Pico**

---

## 🧠 Learning Path

### Phase 1 — C Basics

* Variables
* Functions
* Structs
* Pointers (basics)
* Memory concepts

### Phase 2 — Raspberry Pi Pico

* GPIO
* UART
* I2C
* SPI
* Sensors

### Phase 3 — Embedded Architecture

* Modular design
* Drivers
* Application layer

### Phase 4 — RTOS

* Tasks
* Scheduler
* Multitasking

### Phase 5 — NuttX

* Build system
* Applications
* Drivers
* Shell
